# sam-task-backend

## Como baixar o projeto

### Opcao 1: Baixar via HTTPS (ZIP)
1. Acesse o repositorio no navegador.
2. Clique em **Code** > **Download ZIP**.
3. Extraia o arquivo em uma pasta local.

### Opcao 2: Clonar com Git (HTTPS)
```bash
git clone https://SEU_REPOSITORIO.git
cd sam-task-backend
```

## Instalar dependencias
Depois de baixar/clonar o projeto, execute:

```bash
npm install
```

## Rodar o projeto
```bash
npm run dev
```

Se o seu projeto usar outro script de inicializacao, ajuste conforme o `package.json`.
